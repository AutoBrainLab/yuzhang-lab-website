## 📝 技术细节

### 为什么Hugo会生成重复标题？

**Hugo主题机制**:
```go
// layouts/_default/list.html (简化)
<h1>{{ .Title }}</h1>  <!-- 自动从frontmatter生成 -->

{{ .Content }}  <!-- 用户Markdown内容 -->
```

**用户Markdown**:
```markdown
---
title: "研究资源"  ← Hugo读取这里
---

# 研究资源  ← 用户又写了一次，导致重复
```

**解决方案**: 删除Markdown中的H1，让Hugo自动处理

### 最佳实践

**Hugo Markdown页面结构**:
```markdown
---
title: "页面标题"  ← 只在这里定义一次
description: "描述"
---

简介段落（可选）

## 第一个章节  ← 从H2开始

### 子章节  ← H3
```
